# Pure Texting

Marketing website for Pure Texting - simple SMS/WhatsApp/RCS messaging for small businesses.

Built from the provided Figma mockups. Static HTML/CSS/JS site, no build step required.

## Structure

- site/index.html - main page
- site/css/styles.css - styles
- site/js/script.js - pricing toggle, industry tabs, mobile nav
- design/ - original mockup exports for reference

## Local preview

Open site/index.html directly in a browser, or serve the site folder with any static file server.
